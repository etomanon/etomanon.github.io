Analysis of open geospatial data and the concept of its application

The thesis will analyze various sources of open geospatial data and propose their possible usage. A prototypic web application will be created for chosen data sets. The main idea is to combine two sources: RODOS & Dopravní info. The speed for certain sections of D2 will be gathered by python web crawler (WIP script file here) from RODOS. Dopravní info shares information about traffic via XML files - accidents, road closures and other events. PostgreSQL with PostGIS extension will serve as storage for extracted & transformed data. Finally, GeoServer will be utilized for WMS & WFS (example of loading the data) to display the data in the web map.
